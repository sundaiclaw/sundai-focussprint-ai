# FocusSprint AI

A tiny AI productivity assistant for students and hackers:
- Turn a goal into a 25-minute sprint plan
- Generate 3 concrete tasks
- Add a short reflection at the end

## Why this matters
Most hack builders lose time deciding what to do next. FocusSprint AI gives immediate momentum.

## MVP
- Input: one goal sentence
- Output: 3-step sprint plan + success criteria
- End-of-sprint reflection template

## Stack
- HTML/CSS/JS frontend
- Node.js express backend (mock AI logic)

## Future
- Plug in LLM API
- Save sprint history
- Team mode
